## Идея 1: Industrial Dark (Точный клон оригинала)

**Design Movement**: Industrial Minimalism с тёмной темой

**Core Principles**:
1. Чёрный фон (#0a0a0a) как основа
2. Красный акцент (#dc2626) для активных элементов
3. Строгая сетка с фиксированным sidebar
4. Минималистичные карточки с тонкими границами

**Color Philosophy**: 
- Чёрный фон создаёт профессиональную атмосферу
- Красный акцент привлекает внимание к действиям
- Зелёный для сумм/цен символизирует прибыль
- Жёлтый для имён клиентов выделяет персонализацию

**Layout Paradigm**: 
- Фиксированный sidebar 200px слева
- Основной контент справа с padding
- Карточки в grid-системе

**Signature Elements**:
1. Логотип POLIBEST 911 с красным POLI
2. Красные кнопки действий
3. Таблицы с чередующимися строками

**Interaction Philosophy**: Мгновенный отклик, минимальные анимации

**Animation**: Subtle transitions 150ms для hover-состояний

**Typography System**: 
- System fonts (sans-serif)
- Bold для заголовков
- Regular для текста
